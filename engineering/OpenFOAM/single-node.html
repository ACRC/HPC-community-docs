Single-Node Jobs

Small OpenFOAM jobs can be run on a single node.

Job Script Template

Here is an example job submission script for a single-node OpenFOAM job:

 1#!/bin/bash
 2## Submission script for Cluster
 3#SBATCH --job-name=my_job
 4#SBATCH --time=0-01:00:00 # hh:mm:ss
 5#SBATCH --nodes=1
 6#SBATCH --ntasks-per-node=1
 7#SBATCH --cpus-per-task=1
 8
 9
10
11## system error message output file
12## leave %j as it's being replaced by JOB ID number
13#SBATCH -e my_job_%j.err
14
15## system message output file
16#SBATCH -o my_job_%j.out
17
18# record some potentially useful details about the job: 
19echo Running on host $(hostname) 
20echo Time is $(date) 
21echo Directory is $(pwd) 
22echo Slurm job ID is ${SLURM_JOBID} 
23echo This jobs runs on the following machines: 
24echo ${SLURM_JOB_NODELIST} 
25printf "\n\n\n\n" 
26
27# Load modules required for runtime e.g.
28module load apps/openfoam/6
29
30
31# Run the solver. Take pisoFoam for example:
32
33pisoFoam
34
35echo End Time is $(date) 
36echo "Done pisoFoam finish"
37printf "\n\n"

Note

For further information on the structure and syntax of Slurm job scripts, see the ACRC HPC documentation pages on job types.

How to use

Save the script file in the case folder, cd into the case folder, and submit the script to Slurm.
